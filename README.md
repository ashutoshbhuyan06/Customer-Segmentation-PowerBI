# Customer Segmentation Dashboard using Power BI

## Overview
This project analyzes customer demographics and spending behavior using Power BI. The dashboard helps businesses understand customer segments based on annual income and spending score.

## Features
- Total Customers KPI
- Average Age
- Average Annual Income
- Average Spending Score
- Customer Distribution by Gender
- Age Distribution
- Income vs Spending Score
- Customer Segmentation
- Interactive Filters (Gender and Age)

## Tools Used
- Microsoft Power BI
- DAX
- CSV Dataset

## Dataset
Mall Customers Dataset

## Customer Segments
- High Value Customers
- Medium Value Customers
- Low Value Customers

## Outcome
This dashboard provides insights into customer behavior and supports data-driven marketing strategies.
